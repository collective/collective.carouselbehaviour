Introduction
============

This product allows use collective.carousel for Dexterity Content Types
Just add  follow lines to yours Dexterity Content Type file (profiles/default/types/Your.Content.Type.xml):

::
    <property name="behaviors">
        ...
        <element value="collective.carouselbehaviour.collectivecarouselbehaviour.ICollectiveCarouselBehaviour"/>
    </property>


This product may contain traces of nuts.
